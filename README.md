AlarmClockConsoleApplication(C++)
This project is a console-based Alarm Clock application developed using C++ and Data Structures & Algorithms concepts. The program allows users to add, delete, search, and manage alarms using a simple command-line interface.

Features
Add new alarms
Delete existing alarms
View all alarms
Search for alarms
Sort alarms by time
Real-time alarm checking system
Data Structures Used
Linked List
Linear Search
Bubble Sort
Technologies Used
C++
Data Structures & Algorithms
Windows Sleep API
Console-based interface
How to Run
Compile the program
g++ App.cpp -o App

Run the program
./App
